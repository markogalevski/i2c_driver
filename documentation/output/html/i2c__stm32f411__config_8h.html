Lists all the possible interrupts (and the dma request mode) available to the I2C channel

Enumerator
IT_ERR

IT_EVT 

The I2C raises an interrupt upon events: Start Bit, Address Matching, STOPF, BTF

IT_BUF 

The I2C raises an interrupt when TxE or RxNE = 1 (if IT_EVT is also enabled)

DMA_REQ 

Th2 I2C issues a DMA request upon TxE or TxNE = 1
